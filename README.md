# File-Manager (TotalCommander)
## Funkcjonalności:
```
Wchodzenie do podfolderów i folderu nadrzędnego (jak w total commander) 
Sortowanie plików/ folderów po:
nazwie
dacie utworzenia
Edycja domyślnym programem 
kopiowanie (takie jak w total commander), 
usuwanie
tworzenie nowych folderów
Przeszukiwanie plików po nazwach
wbudowana przeglądarka plików graficznych
wbudowana przeglądarka plików tekstowych
```
